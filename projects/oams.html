Cloud Based Online Attendance Manegement System

  • Technologies: Microsoft Azure , PHP , HTML , CSS , JavaScript , MySQL
  • Github URL: Project Link
  • Application Screenshots: Project Screenshots

Key features include:

User Roles: Seamlessly cater to administrators, lecturers, and students, ensuring a tailored experience for each stakeholder.

Attendance Tracking: Students can mark attendance, view module details, lecture schedules, and track their attendance percentages effortlessly.

Admin Management: Empowering administrators with tools to add, remove, and manage users, generate reports, and oversee system operations.

Lecturer Tools: Providing lecturers with insights into attendance trends, the ability to create attendance forms, and access to pertinent details for efficient class management.

Built using PHP, HTML, CSS, JavaScript, and MySQL, this system exemplifies the fusion of robust backend functionalities with an intuitive frontend interface. Its cloud-based architecture ensures scalability, reliability, and accessibility from anywhere, anytime.
