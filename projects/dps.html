Research Project - Diabetes Prediction System with Most Significant Factors

  • Technologies: Python Flask, HTML, CSS, javascript, NumPy, SQLAlchemy, pandas , NLTK
  • Github URL: Project Link
  • Applicattion Screenshotes: Project Screenshotes

This innovative system employs six machine learning models, including KNN, SVM, and Logistic Regression, to predict diabetes based on significant factors identified through rigorous research. Through meticulous testing and evaluation, we identified the two most accurate models, ensuring reliable predictions for users.

Moreover, the system is designed to continually enhance its accuracy by collecting user testing data and retraining the models accordingly. Security is paramount, with measures such as Google reCAPTCHA and .env file implementation ensuring data protection.

Built with Python, Flask, HTML, and CSS, the project features an intuitive admin interface for seamless management. The AI chatbot enriches user experience by providing instant responses to queries, enhancing accessibility and usability.
